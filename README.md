# deegram Bot
This is TelegramBot & deezer Integration .
Using This One Can Download Songs Via Telegram Bot Using Deegram Library .



[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

## Bot Commands 
<pre> start - Get the Welcome Message \n help - how to use this Bot \n settings - Change your preferences \n info - about bot developer \n stats - about Bot uptime and Storage </pre>

## prerequisites

Api_id -Get This From my.telegram.org
Api_hash - Get This From my.telegram.org
Bot_token - Get this from @BotFather on Telegram
Deezer_token - Get Deezer Arl Cookie using Mozilla and Cookie Manager Extension from deezer.com
